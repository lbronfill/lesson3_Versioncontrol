Learning to code JavaScript statements can not be done without you following the JavaScript Syntax if you do not follow the rules the browser will not be able to understand the commands given. The first rule is that when you are coding you have to remember that JavaScript is case sensitive so the same word can have different context in JavaScript. The second rule is that each statement must end in semicolon. Also to not add extra white space within the statements because JavaScript will ignore them. Third rule is that you can however use comments to add notes to help you or someone else identify or remind what it is that you are trying to do. There are two options when it comes to comments one is that comments can be blocked from displaying on the page by starting with /* and ending with */.The alternative type of comment is the single line comment that would start with // and end with //. Highlight #5 How to code IF statements

An If Statement allows the user to control how that statement will be delivered. Each If statement must start with an IF clause after it can have an else if clause but that is up to the user it becomes optional. If one statement is true then that means that the other must be fake for example if the age of of someone is 18 and can vote then it must mean that anyone else that is not 18 cannot vote, that would be an If statement with an else clause. The use of the If statement, If clause or the else clause is also known as nesting if statements. This applies to web development because it allows for the the computer to understand that if the conditional expression is true then the next statement is false and vice versa. Learning this in web development allows us to give the computer a different actions for different decisions and on how to execute this decisions. As the user the If statements can help us decide if we want the computer to say Good morning if it is before 12 or Good afternoon. It is part of web development that helps the creator command webpages.
